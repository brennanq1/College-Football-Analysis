# 2023 College Football Analysis

## Overview
This project analyzes 2023 NCAA college football team data to identify factors associated with team success and predict total wins. The analysis is conducted in **R** and includes both a standalone script and a reproducible R Markdown report.

## Files
- `college_football_analysis.R` – R script for data loading, cleaning, analysis, and modeling  
- `college_football_analysis.Rmd` – R Markdown report with narrative, visualizations, and results  
- `cfb23.csv` – Dataset used for the analysis (must be in the same directory)
- `College Football Analysis.pdf` - Presentation slides for findings

## Requirements
- R (version 4.0 or later recommended)
- RStudio (recommended)

## Required R Packages
- tidyverse  
- ggplot2  
- dplyr  
- rmarkdown
